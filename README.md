# Course Selling App

A full-stack Course Selling App built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This project allows users to explore, purchase, and access online courses, while admins can manage courses, users, and platform activities efficiently.

---

## Features

### User Panel
- User Registration & Login
- Browse Available Courses
- Search Courses by Category
- Purchase Courses
- Access Purchased Courses
- Watch Course Content
- Update Profile

### Admin Panel
- Admin Login Authentication
- Add New Courses
- Update Course Details
- Delete Courses
- Manage Users
- View Sales Overview
- Dashboard Analytics

### Authentication & Security
- JWT Authentication
- Protected Routes
- Password Encryption
- Role-based Access Control
